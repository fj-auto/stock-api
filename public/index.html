股票数据服务 API

这是一个基于Yahoo Finance的股票数据服务API，提供股票价格、历史数据、公司信息等数据。

API 端点

获取股票历史数据

GET /api/yahoo/history/:symbol

获取指定股票的历史价格数据。

参数：

  • period: 时间范围 (1d, 5d, 1mo, 3mo, 6mo, 1y, 2y, 5y, max)
  • interval: 数据间隔 (1m, 2m, 5m, 15m, 30m, 60m, 90m, 1h, 1d, 5d, 1wk, 1mo, 3mo)

示例: /api/yahoo/history/AAPL?period=1y&interval=1d

获取股票全部信息

GET /api/yahoo/all-info/:symbol

获取股票的所有可用信息，包括基本信息、财务数据、分析师评级等。

示例: /api/yahoo/all-info/AAPL

获取财报日期

GET /api/yahoo/earnings-dates/:symbol

获取公司的财报发布日期。

示例: /api/yahoo/earnings-dates/AAPL

获取完整财报数据

GET /api/yahoo/earnings-full/:symbol

获取公司的完整财报数据。

示例: /api/yahoo/earnings-full/AAPL
